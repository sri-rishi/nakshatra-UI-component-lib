Avatars

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Example of Image Avatar

Avatars with image are available in 4 sizes. You can add class avatar and for size add classes respective to their size avatar-lg-size, avatar-md-size, avatar-sm-size, avatar-xsm-size (e.g. class="avatar avatar-lg-size")
Add Responsive Image class names for img element.

Example of Avatar with Text

You can use initial letters of user in Avatar as well. You can add class avatar and for size add classes respective to their size avatar-lg-size, avatar-md-size, avatar-sm-size, avatar-xsm-size (e.g. class="avatar avatar-lg-size")
